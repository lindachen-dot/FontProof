02
載入字型
1
拖曳字型檔案
將 TTF、OTF 或 TTC 檔案拖曳到左側欄虛線框，或點按框框開啟檔案選擇器。支援一次載入多個檔案。
↑ 拖曳或點按載入字型  ·  TTF  ·  OTF  ·  TTC
2
TTC 自動展開
TTC 檔案內含多個子字型，載入後會自動全部展開顯示於列表中，每個子字型可個別開啟或關閉。
3
啟用 / 停用 / 移除字型
點按右側圓形切換鍵可暫時停用字型；滑鼠移到字型名稱時出現 × 可永久移除。
HFFont-Regular.ttf
×
HFFont-Bold.ttf（已停用）
提示：停用的字型（切換鍵呈灰色）不顯示於檢視區，但顏色和記錄設定仍保留。
4
可變字型（Variable Font）自動偵測
載入含 fvar 表的可變字型時，會自動跳出「Variable Font Instances」視窗，列出該字型的所有座標軸（如 Weight、Width）與可選的 Instance。
Variable Font Instances
Axes: Weight / wght (100–900) · Width / wdth (75–125)
選擇要載入的 Instances：
Regularwght=400, wdth=100
Boldwght=700, wdth=100
勾選需要的 Instance 後按「載入選取」，每個勾選的 Instance 會以獨立字型項目加入列表（名稱會自動加上 Instance 名稱，如「HFFont Bold」），可分別開啟、停用或設定顏色，與一般字型無異。「全選」「全不選」可快速切換勾選狀態，未勾選任一項時無法載入。
03
設定字元集
1
從左側欄下拉選單選擇字元集
開啟工具後預設顯示「選擇字元集」，不自動載入任何字元集。從左側欄的下拉選單選擇後立即套用；選擇「Unicode 範圍」或「載入 Encoding 檔案」時會開啟輸入視窗，可再次點選同一選項重新開啟。
預設字元集
  • ASCII（U+0020–U+007E）
  • MS Windows 1252 Western
  • MS Windows 1250 Central European
  • MS Windows 1251 Cyrillic
  • MS Windows 1253 Greek
  • MS Windows 1254–1258（其他）
  • MS Windows 874 Thai (OEM)
自訂字元集
  • Unicode 自訂範圍（如 U+4E00–U+9FFF）
  • 載入 FontLab Encoding（.enc）
  • 載入純文字字串（.txt）
2
載入 Encoding 檔案格式說明
支援兩種格式，工具自動偵測：
格式 A（.enc）— FontLab Encoding
%%FONTLAB ENCODING: -1; 2026-06-12
%%GROUP:FontProof
uni0041
uni0042
uni0043
格式 B（.txt）— 純文字字串
ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz
0123456789
提示：%% 開頭的注解行會自動略過。再次點選同一選項可重新開啟輸入視窗修改設定。

05
Metrics 疊加

左側欄「Metrics 疊加」區塊，各條線可獨立勾選。點「全開」一次啟用全部，再按變「全關」。

H
Baseline
hhea Ascent
hhea Descent
Typo Ascender
Typo Descender
x-height
Cap Height
LSB / RSB
LSB / RSB（左右側邊距）
  • LSB：字符原點到輪廓最左緣的距離
  • RSB：輪廓最右緣到下一字符原點（advance width 終點）的距離
  • 以兩條灰色虛線標示每個字符格內的字符前進寬度邊界，方便檢查字距是否過寬或過窄
注意：載入多套字型時，所有字型共用統一的 Baseline 位置（取所有字型 hhea 最大值），確保 Metrics 線不超出格子範圍。
06
版面與顯示控制

右側面板「版面設定」頁籤，所有設定均即時套用：

字型大小 24–200 px 調整字符顯示尺寸
字元間距 0–40 px 字符格子之間的水平間距
行距 0–80 px 每行之間的垂直間距
每行字元數 4–40 格狀模式每行排幾個字符

工具列快速切換

框格 顯示 / 隱藏字符格子的邊框。PDF 匯出會跟隨此設定。
記錄標示 開啟時，已記錄字符顯示紅框；關閉時紅框消失，記錄仍保留。
清除全部 移除所有已載入的字型並清除碼位記錄。操作前會出現確認提示，無法復原。

顏色設定

右側面板「顏色」頁籤 → 點按色塊開啟顏色選擇器，可從 12 個預設色直接點選，或輸入 #RRGGBB 自訂顏色。疊加模式的透明度預設 50%，可用滑桿調整。

🔍 Go To 跳轉欄位

工具列右側的搜尋欄，輸入碼位後跳轉到對應字符並以橘黃色邊框高亮標示。

A 直接輸入字元
0041 十六進制碼位
uni0041 FontLab 格式
字符不在目前字元集中時顯示「未收錄」。按 Enter 或直接輸入即可跳轉。
07
碼位記錄
1
雙擊字符格子加入記錄
雙擊任一格子，該字符以 uniXXXX 格式加入記錄，同一字符不重複記錄。若「記錄標示」開啟，格子顯示紅色邊框。

uni0041

2
右鍵單擊字符格子移除記錄
對已在記錄中的字符按右鍵，立即從碼位記錄中移除並取消紅框標示。對未記錄的字符右鍵則不做任何事。
3
管理記錄清單 & 搜尋
右側面板「碼位記錄」頁籤，頂部有搜尋欄可即時過濾清單。點 × 移除單筆；「清除」清空全部。每筆記錄左側顯示字符預覽。

08
匯出 Codebook PDF
1
點按工具列「Codebook PDF」
開啟匯出設定視窗，確認目前字元集、字型大小及框格顯示狀態。
2
選擇版面配置
格狀版面
  • 適合單一字型
  • 字符由左到右、上至下排列
  • 適合輸出完整字元集樣本冊
並排版面
  • 適合多字型比對
  • 同字符各套版本並列顯示
  • 適合版本比對或字型審核
3
點按「產生 PDF」
PDF 以 4× 高解析度 Canvas 渲染，字符清晰不模糊。
PDF 包含內容
✓字符圖形（高解析度）
✓Unicode 碼（uniXXXX）
✓字型名稱
✓字元集與字型大小資訊
框格顯示跟隨介面目前的「框格」開關設定（開框格 → PDF 有框；關框格 → PDF 無框）
建議：匯出前先在介面調好字型大小與框格設定，PDF 會完整反映當前顯示狀態。
